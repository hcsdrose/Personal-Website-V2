# Christopher Rose AI Portfolio

This is a personal portfolio website built with Vue 3, Vuetify, and Vite, inspired by christopherrose.com.

## Features
- Modern, clean design
- Responsive layout
- Sections: Home, About, Projects, Contact
- Vuetify UI components
- Vue Router navigation

## Getting Started

### Install dependencies
```
npm install
```

### Run the development server
```
npm run dev
```

### Build for production
```
npm run build
```

## Customization
- Replace placeholder images and text in `src/views/`
- Update navigation and content as needed
